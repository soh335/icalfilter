# icalfilter

CLI for filtering ical components and lines

## USAGE

```go
$ cat /path/to/ical | icalfilter -filter-component "..." -filter-component "..." -filter-line "..."
```

## INSTALL

```
$ go get github.com/soh335/icalfiter
```
